# Aula de Machine Learning - Predição de Preços (Boston Housing)

Este repositório foi estruturado para a prática de versionamento com Git e GitHub, utilizando um projeto de Machine Learning como base.

## 📂 Estrutura do Projeto

- `notebooks/`: Contém o Jupyter Notebook com a análise e o modelo.
- `data/`: Contém o dataset Boston Housing.
- `requirements.txt`: Lista de dependências Python.
- `.gitignore`: Arquivos que o Git deve ignorar.

## 📊 Dicionário de Variáveis

- **crim**: Taxa de crime per capita.
- **zn**: Proporção de terrenos residenciais zoneados.
- **indus**: Proporção de acres de negócios não varejistas.
- **chas**: Variável dummy do Rio Charles (1 se faz fronteira; 0 caso contrário).
- **nox**: Concentração de óxidos de nitrogênio.
- **rm**: Número médio de quartos.
- **age**: Proporção de unidades construídas antes de 1940.
- **dis**: Distâncias para centros de emprego.
- **rad**: Índice de acessibilidade a rodovias.
- **tax**: Taxa de imposto.
- **ptratio**: Proporção aluno-professor.
- **b**: Proporção de negros por cidade.
- **lstat**: Porcentagem de status inferior da população.
- **medv**: Valor mediano das casas (Alvo).

## 🚀 Como começar

1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Explore o notebook em `notebooks/`.
